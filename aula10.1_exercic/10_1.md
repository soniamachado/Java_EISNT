exerc10.1 Classe Animal (superclasse)
Cria uma classe Animal com:
•	nome, idade
•	emitirSom() → método genérico

Depois cria 2 subclasses:
Cao → sobrescreve (fazer override) emitirSom() para "Au au!"
Gato → "Miau!"
Tarefas:
•	na classe Cao → sobrescreve (fazer override) emitirSom() para "Au au!"
•	na classe Gato → sobrescreve (fazer override) emitirSom() para "Miau!"
•	Criar uma classe Main onde testem o metodo emitirSom() tanto no Gato e no Cao
