Criar um programa em Java que leia um ficheiro CSV com informação de empregados, crie os respetivos objetos em memória e calcule:

O salário total mensal
A banda de IRS com base no rendimento anual (14 meses)

Este exercício pretende praticar:

Programação orientada a objetos
Herança
Leitura de ficheiros
Uso de listas (ArrayList)

📄 Ficheiro CSV
O programa deve ler um ficheiro com o seguinte formato:

 
Nome,TipoEmpregado,SalarioBase
Ana,FULLTIME,1200
Pedro,HALFTIME,800
Maria,FULLTIME,1500
Joao,HALFTIME,900
 
Campos:
Nome – nome do trabalhador
TipoEmpregado – pode ser FULLTIME ou HALFTIME
SalarioBase – salário base mensal



🏗 Estrutura do Programa

👨‍💼 Classe Empregado
Criar uma classe base chamada:

Empregado
 
Atributos
String nome
double salarioBase

Métodos
Construtor
Getters e Setters

Método:
double calcularSalarioTotal()
double calcularBandaIRS()
Este método pode/deve ser sobrescrito nas subclasses.

👨‍💻 Classe EmpregadoFullTime
Esta classe deve herdar de Empregado.
Regra de salário
O empregado Full Time recebe: 
salarioTotal = salarioBase + 100
Deve implementar o método:
calcularSalarioTotal()
 
👩‍💻 Classe EmpregadoHalfTime
Esta classe também deve herdar de Empregado.

Regra de salário
O empregado Half Time recebe: 
salarioTotal = salarioBase + 25
Deve implementar o método:
calcularSalarioTotal()
 
💰 Método para calcular Banda de IRS
Criar um método que calcule a banda de IRS com base no rendimento anual.

Fórmula
rendimentoAnual = salarioTotal * 14
 
Tabela simplificada de IRS
Rendimento Anual	Banda IRS
até 10 000€	0%
10 001€ – 20 000€	10%
20 001€ – 35 000€	20%
mais de 35 000€	30%
Criar o método na classe Empregado (este nao necessita de ser feito override):
double calcularBandaIRS()
 
📂 Leitura do CSV
Criar uma classe responsável por:

LeitorCSV
Este método deve:
Ler o ficheiro CSV
Ignorar a primeira linha (cabeçalho)
Criar os objetos corretos:
EmpregadoFullTime
EmpregadoHalfTime
Em cada iteração deve mostrar a informação de cada empregado
 
▶️ Classe Main
Na classe principal devem:
Ler o ficheiro CSV
Para cada empregado mostrar: 
Nome
Tipo
Salário Base
Salário Total
Rendimento Anual
Banda IRS
 
Exemplo de saída: 
Nome: Ana
Tipo: FULLTIME
Salário Total: 1300
Rendimento Anual: 18200
Banda IRS: 10%
 
⭐ Desafio Extra (Opcional)
Para quem terminar:
Guardar os Empregados todos num array to tipo
ArrayList<Empregado>
Calcular o salário médio da empresa
Mostrar quantos empregados existem de cada tipo
Ordenar os empregados por salário anual
